求n值的简洁证明：

由于1+2+3+4+...+n= n*(n+1)/2，即前面推导过程中的条件等同于n*(n+1)/2 刚好大于或等于target
换算一下可得 2 * target <= n * n + n， 由于也有target >= n*(n-1)/2 ，即2 * target >= n * n - n

可获得 n * n + n >= 2 * target >= n * n - n 形式的双联不等式，

由于n * n + n 可变换为n * n + n / 2 + n / 2, 即(n + 1/2)的平方,
同理，n * n - n 可变换为(n - 1/2)的平方。

对双联不等式同时开方，就会看到 2 * target 开方后的值，落在n + 0.5 和 n - 0.5的区间内。 所以可以用四舍五入法对2* target的开方直接取n值。

另外由于2 * target为整数，而n + 0.5或n - 0.5 的平方必然不是整数，所以四舍五入法不会发生区间边界重合的情况，也就必然不可能因为边界重合导致错误的四舍五入。

证明完毕。
